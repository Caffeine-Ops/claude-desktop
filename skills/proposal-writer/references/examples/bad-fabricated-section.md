# 范例 · 编造的坏小节（反例，别学）

下面是**典型的编造**，逐处标出为什么错、该怎么改。

## 坏示例

```markdown
##### 智能诊断

本系统采用业界领先的自研深度学习大模型，诊断准确率高达 99.6%，
性能全面超越 GPT-4。系统可自动生成诊断结论并一键开具处方，
大幅提升医院效率 200%，患者满意度达 100%。

![诊断界面](./images/diagnosis.png)

（据《产品说明》）
```

## 逐处病灶

- **「智能诊断 / 自动生成诊断结论并开具处方」**：素材里没有这个功能，且 AI 直接诊断开药**不合规**——凭空编造功能。
- **「业界领先的自研深度学习大模型」「性能全面超越 GPT-4」**：无出处的吹嘘。
- **「准确率 99.6%」「效率 200%」「满意度 100%」**：编造的数字，客户据此采购风险极大。
- **`![诊断界面](./images/diagnosis.png)`**（独立模式）：**编造的图片路径**，违反接地底线。
- 末尾虽标了`（据《产品说明》）`，但内容根本不来自它——**假溯源比不标更糟**。

## 应该怎么写

- 素材没有"智能诊断"功能 → **根本不写这节**（不编功能）。
- 若素材有"预问诊报告生成"（辅助，不是诊断）→ 写那个真实功能，用真实说明。
- 所有数字只在有素材出处时写；没有就不写或标：
  `⚠️ 资料缺失：预问诊报告的准确率/满意度等量化指标（素材未提供）`
- 独立模式配图写占位：`〔建议插入：预问诊报告界面截图〕`，不编路径。
