输出面板 · 重设计

上一版新增动效的问题：整行铺一层品牌色底色再渐隐，读起来像警告闪烁而不是"新增"的欣喜；文件名旁插一个 文字标签，标签出现/消失各挤一次文字截断宽度，两次布局抖动叠加显得毛躁。这版把"新在哪"这件事从"整行强调" 收窄成三种克制程度不同的局部提示，颜色/字体/圆角/缓动全部取自 design-tokens 与现有卡片配方， 可用上方「新增动效」切换实时对比。

01重设计后的输出面板

三档新增动效共享同一条位移淡入底座（back-out 缓动，不带 scale 抖动）：极简 只有这层底座， 不额外强调"这行是新的"，交给用户自己扫一眼列表顶部；左侧强调条 借用 Linear「新动态」的语言， 细条扫入再淡出，不碰行内背景、不占用文字空间；角标圆点 是 app 图标未读角标的通用语汇，落在 缩略图内角脉冲两下后常驻淡出。触发按钮本身统一是双层同心圆提示环（圆形而非方框描边放大，方框缩放 容易读成 UI 报错）+ 数字徽标弹出，面板关着也能被余光注意到。
