AskUserQuestion · 替换 composer 的状态切换

提问到达时 composer 整个变形为提问面板（同一张卡：高度 FLIP + 内容错拍 crossfade + 圆角/描边/阴影渐变）， 答完/取消再变回来——提问态下不存在输入框，自由回答走面板内的「其他」行。

状态机：idle（composer）⇄ asking（提问面板）。工具请求到达 → morph 进 asking；全部作答 / 全部跳过 / 点 × 取消 → morph 回 idle， 答案回传 AskUserQuestion，消息流 tool 行同步翻「已回答」。
morph 规格：容器高度 FLIP 340ms cubic-bezier(.32,.94,.35,1)；旧面 180ms 上飘淡出（absolute 钉住不占高）， 新面延迟 100ms 下方浮入；圆角 22↔18px、绿描边 ↔ 大投影同步过渡。题间切换（1/2 → 2/2）同样走高度 FLIP。
键盘：1–9 直选 · Esc 跳过当前题 · 输入态 Enter 发送。 ‹ › 可回看已答题（绿勾圈，可改答案）；最后一题跳过 = 提交已答部分。
