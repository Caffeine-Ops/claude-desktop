现状的三个问题：宽窗口下 14px 标题孤零零漂在左半区，读作调试信息而非会话锚点；「AI 生成」hairline 胶囊带边框、存在感盖过标题本身；标题与正文字号几乎无层级差。以下四案从最稳（macOS 原生语言）到最激进 （无界沉浸）排列，色彩与字体全部取自 design-tokens，顶栏 46px 布局契约不动（04 为 48px）。
