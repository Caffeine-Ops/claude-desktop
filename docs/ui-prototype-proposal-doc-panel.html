一、系统概述

智慧导诊与预问诊系统面向医院门诊全流程，覆盖患者「来院前—院中—诊前」三个环节。导诊系统通过智能问答帮助患者快速定位科室与医生；预问诊系统在候诊阶段完成病史采集，生成结构化预问诊报告推送给接诊医生，缩短单次接诊时间。

本方案依据贵院信息化建设规划，结合两个产品的知识库资料编制，功能描述均以产品白皮书与实施手册为准，未经证实的内容不作展开。

二、系统功能架构

2.1 总体架构

系统采用前后端分离的微服务架构，前端覆盖微信公众号、小程序与院内自助机三类入口，后端由分诊规则引擎、医学知识图谱服务与报告生成服务组成，统一经院内集成平台与 HIS、排班系统对接。

资料缺失：预问诊系统与 HIS 的对接接口规格（协议、字段映射）在知识库中未找到

2.2 关键技术

  • 医学知识图谱驱动的多轮问诊对话，症状—科室映射可由医院自行维护；
  • 分诊规则引擎支持症状、科室、排班三维联合匹配；
  • 结构化报告生成，预问诊结果按病历书写规范输出主诉与现病史。
